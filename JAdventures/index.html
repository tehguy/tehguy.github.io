About

JAdventures was made using Scott Lilly's simple C# RPG tutorial and converting all the C# into Java just because I can. Of course, it wasn't enough for me to just convert and call it a day so I took it upon myself to toss in a couple of things I figured would be improvements.

Features

Feature Since
Entity/Location/Object data saved as JSON files
Visual character creator
D&D 5e-like stat handling
D&D 5e classes
