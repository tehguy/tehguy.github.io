About

Dungeon Adventures was the original attempt using Scott Lilly's simple C# RPG tutorial as a base and then kinda stacking my own things on top of it such as the D&D 5e ruleset and a character generator based on said ruleset (which is in the latest build). This project has been superseded by JAdventures and wont be worked on any more nor will it be maintained any further.
